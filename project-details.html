Wind Turbine Manufacturer Client

B2B E-commerce
platform

  •   Duration

    10 months

  •   Domain

    Manufacturing

  •   Role

    UX design, UI design

OVERVIEW

A platform for selling various
categories of wind turbine parts.

Wind turbine manufacturing client, who is a leading supplier of wind power solutions to customers all over the globe. The client team reached out to us with the problems they have in their legacy system and they wanted us to provide a digital solution to sell their wind turbine products. Through which their customers should be able to easily browse, request, track and record the purchased products.

PROBLEMS

Some of the problems with the
existing system:

01

There was no option exists for users to
view the products and their features
online.

02

Hectic for the users to know about the
new alternative products available.

03

The bulk ordering of products and tracking
them was not that easy for the users.

04

Tracking order history was a heavy time-
taking process with the existing system.

05

The manual process of handling restrained
products is time taking.

USERS

Executive

Approver

We have gathered information about the users and their process of ordering products in the existing system. We synthesize the data we have collected from the stakeholders and extract the insights, which helps us move forward in this design process.

How did we help resolve the
problem

A B2B e-commerce platform, which has the powerful ability to market the products and let the users browse, request, order, etc. we help solve the problems with some of the implementations.

Request

Allows the user to request a product, which is not available in the specified category.

Bulk order

Allows the user to order in large quantities with a help of an excel template.

Back order

This helps the user get notified about the restraining orders and assists to take action instantly.

Alternative to obsolete

Suggest the new alternate products when the product the user wishes to buy is obsolete.

Image search

Allow the user to search for the product with the help of image content.

100+ Screens

No. of Web application screens delivered.

CONCLUSION

The process of viewing products, getting to know of the new alternatives, handling restrained products was restless for the users before but these implementing design solutions can solve the user's problems and make the sale more effective.
